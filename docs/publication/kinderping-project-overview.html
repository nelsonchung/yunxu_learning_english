CareLink 安親班接送協作平台

CareLink 以單一 App 打通管理者、老師與家長的接送流程，提供即時狀態同步、角色權限控管與推播提醒，協助園所把繁瑣點名作業變成數位化流程。

安親班常見痛點

  • 接送訊息散落在紙本或社群軟體，老師難以同步給所有夥伴。
  • 缺少統一帳號系統，臨時調班或增設分校常常得重新建立名單。
  • 家長無法掌握孩子是否已準備好，延誤接送或造成櫃台電話塞車。

CareLink 三大支柱

統一身份與資料模型

透過 Firebase Auth 與 Firestore 建立管理者、老師、家長與學生的關聯資料，登入即載入個人所屬班級與孩子。

即時接送與狀態看板

老師可更新學生狀態、建立接送請求；家長端即刻收到最新資訊，避免反覆來電確認。

自動化通知與紀錄

Cloud Functions 推播通知家長孩子準備情況，所有互動留存於 Firestore，方便稽核與追蹤。

CareLink App 三大操作情境

Demo 實際畫面，協助家長、老師與決策者在幾分鐘內掌握產品互動流程。

家長註冊

以手機號驗證身分，簡單，快速，好上手。

登入畫面

管理者、老師、家長使用同一套 App，登入後自動套用 Firestore 權限，僅看見與自己相關的班級與任務。

管理員管理儀表板

可以設定學生，家長，老師與班級的關係，方便管理狀態。

多角色儀表板

管理者快速新增老師與家長帳號；老師掌握班級狀態；家長查看孩子接送紀錄，一套系統滿足三個角色。

Firebase 即時同步

使用 Provider + ChangeNotifier 串接 Firestore 串流，無論在手機或平板都能看到最新接送狀態。

推播通知流程

sendPickupNotification Cloud Function 依角色驗證並推送 FCM 訊息，確保家長第一時間收到提醒。
